Linux系统

Linwux是一套免费使用和自由传播的Unix操作系统,是一个基于POSIX(可移植操作系统借口)和Unix的多用户,多任务,多线程和多CPU的曹祖欧系统

目录结构

目录 说明
/ 根目录
/bin 村饭的是Linux时使用的一些核心文件
/dev 存放Linux的外部设备
/etc 存放所有的系统管理所需的配置文件和子目录
/home 用户的主目录,每个用于都有自己的目录
/lib 存放系统最基本的动态连接共享库
/lost+found 当系统非法关机后,存放一些文件
/media 自动识别一些设备
/mnt 系统提供该目录是为了让用户临时挂载别的系统文件
/opt 给主机额外安装软件摆放的目录
/proc 这是一个虚拟目录,他是系统内存的映射
/root 系统管理员的目录
/sbin 存放系统管理员使用的管理程序
/srv 存放一些服务启动之后需要提取的数据
/sys 2.6内核中出现的文件系统sysfs
/tmp 用来存放临时文件的
/usr 用户很多程序和文件都放在这个目录下
/usr/bin 系统用户使用的应用程序
/usr/sbin 超级用户使用的比较拥挤的管理程序和系统守护程序
/usr/src 内核源代码存放目录
/var 这个目录存放不断扩充的东西
/run 临时的文件系统

常用命令

目录管理

绝对路径 :从根目录开始

相对路径 :从当前文件目录开始

命令 说明 举例
cd 切换目录
.. 代表上一级目录
./ 代表当前目录
/ 代表根目录		 cd /
ls 列出当前目录下所有文件和文件夹
-a 查看所有文件,包含隐藏文件
-l 列出所有文件,包含文件属性和权限,没有影藏文件		 ls -a
pwd 返回当前用户所有目录 pwd
mkdir 创建目录
-p 创建层级目录		 mkdir test
rmdir 删除目录
-p 代表强制删除		 rmdir test
cp 复制目录 cp test .txt ./test
rm 删除文件/目录
-f 忽略不存在的,强制删除
-r 递归删除目录		 rm test.txt
mv 移动文件或目录
-f 强制
-u 只替换跟新过的文件		 mv test ./test

基本属性

Linux系统是一个典型的多用户系统,不同用户处于不同的地位,拥有不同的权限.为了保护系统安全性.Linux对不同的用户访问同一文件的权限做了不同的规定

第一个字符

  • d 代表是目录
  • - 代表是文件
  • l 代表连接文档
  • b 代表装置文件里面可随机存取的装置
  • c 代表装配文件里串行端口设备

接下来的字符3个为一组

  • r 代表可读
  • w 代表可写
  • x 代表可执行
  • - 代表没有权限

设置文件属性的两种方式:数值和符号

  • r : 4
  • w : 2
  • x : 1
命令 说明
chmod 777 filename 所有用户获取读写执行
chmod rwxrwxrwx filename 所有用户获取读写执行

文件命令

命令 说明
cat 从第一行开始显示文件内容
tac 从最后一行开始倒着显示内容
nl 显示的时候,输出行号
more 一页一页显示文件内容
空格代表翻页,回车代表向下看一行
less 一页一页显示,可以向前看
空格代表翻页,上下键代表上下看一行,q代表退出
在查看的时候,使用 /搜索字符 来搜索指定内容
head 只看头几行
通过-n 来控制几行
tail 只看尾几行
tar 打包或解压
tar -zxvf 压缩文件.tar.gz 解压
tar -zvvf 压缩文件名.tar.gz 源文件

链接

链接分为硬链接和软连接

  • 硬链接 :b是a的硬链接,他们指向的都是同一个文件,如果a删除了,b还是可以访问到原始数据
  • 软链接 :类似快捷方式,b是a的软链接,如果a删除了,b也就不能用了
命令 说明 举例
ln 创建硬链接 ln a b
ln -s 创建软链接 ln -s a b

Vim编辑器

vim编辑器是vi发展出来的文本编辑器,如果文件存在,就修改该文件,不存在,就创建

分为命令模式,输入模式和底线命令模式

  • 命令模式 :用户启动vi/vim,进入命令模式
    • i :切换输入模式
    • : 切换底线模式
    • / 需要查找的字符串 :查找字符串
  • 输入模式
    • 回车键 :换行
    • 删除键 :删除
    • home/end键 :光标移动到首行或尾行
    • pgup/pgdn键 :撒很难过I型啊翻页
    • esc键 :退出输入模式,进入命令模式
  • 底线模式
    • q :退出程序,!代表强制退出
    • w :保存文件
    • :set nu :设置行号

用户

命令 说明 举例
useradd 创建用户名
-m 自动创建用户的主目录
-G 分配组		 useradd -m test
userdel 删除用户
-r 自动删除用户的目录		 userdel -r test
su 切换用户 su root
password 设置密码 password test
passwd 锁定账户
-l 锁定用户
-d 清空用户密码
-u 解锁用户		 passwd -l test

进程管理

命令 说明 举例
ps 查看当前系统中正在执行的信息
-a 显示终端运行的所有的进程信息
-u 以用户的信息显示进程
-x 显示后台进程的参数		 ps -a
| 管道符,将前面的结果当做后面的参数传入 ps -aux | grep mysql
grep 查找文件中符合条件的字符串 ps -aux | grep mysql
pstree 查看进程树
-p 显示父id
-u 显示用户组		 pstree -pu
kill 结束进程 kill -9 3323
